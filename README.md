# endo

Yusuke Endo — portfolio. 遠藤佑亮 ポートフォリオサイト。

実写映像制作・グラフィックデザイン・展示デザイン。

Static site (vanilla HTML/CSS/JS). Entry point: `index.html`.

## Local

Just open `index.html` in a browser, or serve the folder:

```
python3 -m http.server
```

## Deploy

Static — no build step. Hosted on Vercel (Framework Preset: **Other**, output = repo root).
